# aur.nim

A nim client for the [Arch User Repository](https://aur.archlinux.org/) (AUR).

## Installation

Add the following to your `.nimble` file:

```
# Dependencies

requires "aur >= 0.1.0"
```

Or, to install globally to your Nimble cache run the following command:

```
nimble install aur
```
